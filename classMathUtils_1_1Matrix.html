Detailed Description

template<std::size_t ROWS, std::size_t COLS>
class MathUtils::Matrix< ROWS, COLS >

2D matrix class.

Stores elements in row-major order. All elements are zero at initialization.

Template Parameters
ROWSNumber of rows.
COLSNumber of columns.
